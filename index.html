Video Overlay

1. Select video

Select a video file from your disk. All your data stays on your computer!

2. Select data

CSV table should include minimum 2 columns, the first should be time. Table elements should be separated with ";": 
# data.csv

time; var1; var2
0; 0; 1
0.1; 0.099833417; 0.995004165
0.2; 0.198669331; 0.980066578
0.3; 0.295520207; 0.955336489
0.4; 0.389418342; 0.921060994

3. Select configuration

Configuration is done via yaml files. Each top level element is a column in the csv file: 
# config,yaml

var1:
    unit: m/s
    style:
        - left: 50px
        - top: 250px
        - font-family: "Courier"
    
var2:
    unit: m/s
    style:
        - left: 250px
        - top: 250px
        - font-family: "Arial"
        - font-size: 250px
